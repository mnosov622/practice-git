# ✏️ Personal Blog Website
This is a front-end template for a personal blog website. Feel free to either use the template or contribute towards the repository. 

## ⚒️ Technologies Used:
<ul>
<li>HTML5</li>
<li>CSS3</li>
<li>JavaScript</li>
<li>Normalize.css</li>
<li>BootStrap/BootStrap Icons</li>
</ul>

## 📁 Folder Structure:
